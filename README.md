# Trading-Company
Practise task.
This project is example of backend app for trading company.
***
# Abilities
- Add/edit clients
- Add/edit employees
- Add/edit products
- Add/edit orders (the client and employee who processed the order)
- Add/edit order info (number of products)
- Add/edit type of product
***
# Using:
- Java 17
- Spring boot
  - Spring data JPA
  - Spring validation
  - Spring pagination
- MySQL
- Liquibase
